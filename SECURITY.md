# Security Policy

## Reporting a vulnerability

If you discover a security vulnerability, do not open a public issue.

Please report it privately to:

- `root@kuchizu.com`

Include:

- a short description of the issue
- steps to reproduce it
- affected components
- potential impact

If possible, also include:

- logs
- screenshots
- proof of concept

## Disclosure

Please allow reasonable time to investigate and fix the issue before public disclosure.
